# Style Guide: Produsom

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

> 💡 These are just the design sizes. The content is responsive and meets WCAG requirements by testing the full range of screen sizes from 320px to large screens.

## Colors

### Brand Colors

- Prussian Blue: (#1b354b) hsl(210, 46%, 20%)
- Citrine: (#efd514) hsl(53, 91%, 50%)

### Neutral

- Light: `light` (#f8fafc)
- Medium: `medium` (#94a3b8)
- Dark: `dark-prussian` (#020617)

## Typography

- Family: [Inter](https://fonts.google.com/specimen/Inter)

## UI Elements

- **Buttons:** Rounded corners (`rounded-md`), hover state with brightness increase.
- **Cards:** Image background, subtle dark overlay `bg-slate-900/60`, hover state with overlay 0%
- **Transitions:** Smooth 200ms ease-in-out for all hovers.
