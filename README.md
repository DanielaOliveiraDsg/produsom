# PRODUSOM - Visual Identity & Front-End Concept

![A full-cycle project from Brand Identity to Responsive Web Implementation.](jpg)

## Welcome! 👋

Thanks for checking out the project.
In 2023 I originally developed the visual identity for the company Produsom. Recognizing a need for a digital presence beyond social media, I developed this web concept to bridge the gap between their physical branding and digital client acquisition.
Produsom is a Brazilian company that provides lighting and sound solutions for events.
